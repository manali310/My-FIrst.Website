Full-Stack E-Commerce Platform

Project Overview

This is a standard industry-level Full-Stack E-Commerce web application built using the MERN stack. It allows users to browse products, add items to a dynamic shopping cart, and complete secure payments online. It also features a fully functional Administrative Dashboard for managing orders and product inventory in real-time.

Technologies Used

MongoDB Express.js React.js Node.js JWT Auth Stripe API

Key Features Implemented

  • User Authentication: Secure signup/login using JWT and bcrypt hashing.
  • Product Filtering: Dynamic search, category sorting, and price range filters.
  • Secure Checkout: Integrated Stripe/Razorpay payment gateway.
  • Admin Dashboard: Real-time product creation, deletion, and order status updates.

Core Backend Implementation (Code Snippet)

Here is a preview of the backend architecture structure designed for this platform:

models/Product.js
const mongoose = require('mongoose');

const productSchema = new mongoose.Schema({
  name: { type: String, required: true },
  price: { type: Number, required: true },
  category: { type: String, required: true },
  stock: { type: Number, default: 0 }
}, { timestamps: true });

module.exports = mongoose.model('Product', productSchema);
routes/payment.js
const express = require('express');
const router = express.Router();
const stripe = require('stripe')(process.env.STRIPE_SECRET_KEY);

router.post('/checkout', async (req, res) => {
  const session = await stripe.checkout.sessions.create({
    payment_method_types: ['card'],
    line_items: req.body.items,
    mode: 'payment',
    success_url: 'http://localhost:3000/success',
  });
  res.json({ id: session.id });
});
